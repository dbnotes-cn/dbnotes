应用软件安装·版本指导

关于应用软件版本的选择，推荐使用官方、最新的LTS版（长期支持版本）。

注：LTS（Long-Term Support）版本被认为是最稳定的版本，它经历了广泛的测试，并且大多包含了多年积累的改进。

  • 本素材由好友 yoyo 友情提供～

操作系统

  • 关于操作系统的选择，考虑到 CentOS 8 已于 2021-12-31、CentOS 7 已于 2024-06-30 停止维护，企业服务则推荐 RedHat Enterprise Linux 8.6（有授权费用），免费系统则推荐 Oracle Linux 8.10，阿里云则推荐 Alibaba Cloud Linux 3.2104 LTS
  • 信创场景，推荐 银河麒麟 Kylin Server V10 SP3，若华为服务器则推荐 EulerOS 2.5
  • 免费场景，推荐 Ubuntu 24.04.3
