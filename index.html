🎯 关于本站

dbnotes.cn 是一个专注于数据库技术与系统运维的技术博客，致力于分享实战经验、技术心得和解决方案。 本站内容涵盖从传统关系型数据库到国产数据库、从Centos/Redhat系统到国产KylinV10/UOS20系统、从 NewSQL 到K8S及云原生架构的全方位技术栈，为 DBA、运维工程师、IT技术爱好者提供有价值的技术参考。

📁 技术栈详情

数据库技术栈

1. 老牌数据库

Oracle - 企业级数据库管理、性能优化、高可用架构、RAC 集群、Data Guard 容灾、备份恢复策略

SQL Server - SQL Server 安装配置、HA CLUSTER集群、AlwaysOn 可用性组、性能调优、备份恢复策略

MySQL - MySQL 主从复制、InnoDB 引擎优化、慢查询分析、索引优化、分库分表方案、备份恢复策略

PostgreSQL - PG 高级特性、扩展插件、逻辑复制、分区表、性能监控、备份恢复策略

2. 国产数据库

DM8 (达梦) - 达梦数据库部署、数据守护集群、DSC集群、迁移方案、性能优化、运维管理、备份恢复策略

KingbaseES9 (人大金仓) - 金仓数据库实战、兼容性适配、集群部署、备份恢复策略

OceanBase (奥星贝斯) - 分布式架构、Paxos 协议、多租户管理、容灾备份

TiDB (平凯) - HTAP 架构、TiKV 存储、PD 调度、SQL 优化、扩缩容实践

YashanDB (崖山) - 崖山数据库特性、兼容性、性能测试、备份恢复策略

OpenGauss (华为) - 高斯数据库部署、WAL 日志、逻辑复制、安全特性

3. NewSQL 类

Redis - 缓存数据库、集群部署、持久化方案、性能优化、分布式锁 、缓存穿透/雪崩/击穿解决方案 MongoDB - 文档型数据库、副本集、分片集群、索引优化、聚合查询

Neo4j - 图数据库、Cypher 查询、图算法、知识图谱应用

InfluxDB - 时序数据库、数据采集、连续查询、监控存储方案

ElasticSearch - 全文检索、倒排索引、集群管理、查询优化、ELK 技术栈

系统技术栈

1. Linux 系统相关

Linux 系统管理 - CentOS、Ubuntu、KylinV10 等主流发行版的安装配置、内核调优、用户权限管理、系统安全加固

日常维护技巧 - 日志分析、故障排查、性能监控、磁盘管理、网络配置、服务管理

2. Shell 脚本相关

Shell 编程 - Bash 脚本开发、自动化运维脚本、日志处理、数据备份、监控告警脚本

最佳实践 - 脚本规范、错误处理、性能优化、调试技巧

3. Docker 技术相关

容器化实践 - Docker 镜像构建、容器编排、网络配置、存储卷管理

生产环境应用 - Docker Compose 多容器编排、镜像仓库搭建、安全加固、性能优化

4. K8S 运维相关

Kubernetes 集群管理 - K8s 集群部署、Pod/Deployment/Service 管理、StatefulSet 有状态应用

运维实战 - 滚动更新、回滚策略、资源配额、命名空间管理、RBAC 权限控制

排错与调优 - 日志收集、故障诊断、性能监控、网络策略、存储类管理

生态工具 - Helm 包管理、Kustomize 配置管理、Prometheus 监控、Istio 服务网格

5. 中间件相关

配置中心 - Etcd 集群管理、键值存储、服务发现、分布式锁 负载均衡 - Nginx 反向代理、负载均衡算法、SSL 配置、性能优化

服务治理 - Nacos 注册中心、配置中心、服务发现、健康检查

消息队列 - Kafka 集群部署、消息可靠性、消费者组、性能调优、RabbitMQ 消息路由

对象存储 - MinIO 部署、存储桶管理、访问控制、数据迁移

6. 监控与告警

Prometheus - 指标采集、PromQL 查询、告警规则、服务发现、Pushgateway

Grafana - 可视化仪表盘、图表配置、告警通知、模板管理

Zabbix - 主机监控、自定义监控项、触发器、自动发现、分布式监控

日志分析 - ELK Stack (Elasticsearch + Logstash + Kibana)、Fluentd 日志收集、日志聚合分析
